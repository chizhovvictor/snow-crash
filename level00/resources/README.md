Level00
=======

Flag
----

x24ti5gi3x0ol2eh4esiuxias

Discovery
---------

Для начала необходимо найти файлы, связанные с пользователем `flag00`. Используем:

    find / -user "flag00" 2>/dev/null

2>/dev/null — перенаправление потока ошибок, чтобы очистить вывод от сообщений об ошибках (например, "Permission denied").
Вывод команды:

    /usr/sbin/john
    /rofs/usr/sbin/john

Оба файла содержат строку:

    cdiiddwpgswtgt

Полученная строка `cdiiddwpgswtgt` не подходит как есть для входа под flag00. Вероятно, это зашифрованная строка. Возможные методы дешифровки:
*   Caesar Cipher
*   ROT
*   Base64 Encoding
*   XOR
*   Простая замена символов
*   Анаграммы

Также полезны сайты вроде https://www.dcode.fr/, где можно попробовать различные алгоритмы дешифровки.

На форумах (GitHub, dcode.fr) были протестированы несколько методов:
*   Chiffre ROT (Rotation)
*   Code César
*   Codage Base62
*   Substitution Mono-alphabétique
И др.

Методом перебора выяснилось, что это Caesar Cipher со смещением, и в первом варианте декодирования был получен пароль:

    nottoohardhere

Это и оказалось паролем пользователя `flag00`.

Use
---

Вход под пользователем flag00 с найденным паролем:

    su flag00
    Password: nottoohardhere

После успешного входа можно получить флаг уровня.

Prevention
----------

Какие меры безопасности предотвращают подобные случаи на реальных системах:

*   **Не хранить пароли в открытом виде** — даже если они размещены в закрытых директориях.
*   **Использовать современные криптографические методы**, а не простые шифры (Caesar/ROT13).
*   **Ограничивать доступ к файлам по принципу минимально необходимых привилегий**.
*   **Проводить регулярный аудит файловой системы** на предмет подозрительных данных.
*   **Использовать контроль целостности** (AIDE, Tripwire) для обнаружения нежелательных изменений в системных файлах.

Documentation
-------------

Методы, использованные в решении:

*   **Caesar Cipher** — шифр с циклическим сдвигом каждой буквы алфавита.
*   **ROT** — разновидность Caesar, где сдвиг фиксирован (например, ROT13).
*   **dcode.fr** — онлайн‑инструмен для дешифровки, перебора сдвигов, анализа частот.
*   **`find`** — поиск файлов по атрибутам файловой системы.
